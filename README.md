# rust-simple-api
(WIP) Test Rust Rocket and Diesel

**Commands**:
  - Run in dev 
    - `cargo run`
  - Build 
    - `cargo build --release`
  - Run after build
    -  `./target/release/rust-simple-api`
  - Docker build 
    - `docker build -t rust_simple_api .`
  - Docker run
    - `docker run -p 8080:8080 rust_simple_api`
